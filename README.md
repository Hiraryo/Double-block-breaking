# Double-block-breaking
大学1回生の基礎プログラミング演習の課題用に作ったゲームです。

### 見た目は
## ブロック崩し
### でも遊び方は普通じゃ無い！

このブロック崩しは、上下にあるバーを操作してブロックを全滅させましょう！

上下のバーはお互い左右反転して移動します。

頑張って！
